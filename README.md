# System monitor plasmoid
Simple plasmoid for monitoring CPU and RAM

# Installation
Download the zip in the releases tab

`unzip cpu-ram.zip`

Move to the plasmoids folder

`mv cpu-ram ~/.local/share/plasma/plasmoids/cpu-ram`
