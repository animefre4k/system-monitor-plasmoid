# System monitor plasmoid
Simple plasmoid for monitoring CPU and RAM

# Installation
Download the project as a zip and extract it

`unzip system-monitor-plasmoid-main.zip`

Move to the plasmoids folder

`mv system-monitor-plasmoid-main ~/.local/share/plasma/plasmoids/cpu-ram`
